Check if a Credit Card is Valid

Using Luhn Algorithm, we can check whether a card is valid. Code is available in Python and C.

This is purely javascript so no information is stored anywhere once you refresh your browser.


The code verifies with built in logic based on credit card culture of VISA, MASTERCARD and AMERICAN EXPRESS

I stumbled upon a youtube video where I saw someone do this in C language. I got inspired and thought that this would be a great project get hands on experience. I spent a good time looking at the video and following along to write the program in the same language.

Rewriting in Javascript and Python

I think this was great learning experience cross language overall
